# Flutter To-Do App
A clean and modern **To-Do application built with Flutter** using Material 3 design.
This app supports task completion, swipe-to-delete, persistent local storage, and
automatic light/dark mode based on system settings.
---
##Features
- Add new tasks
- Mark tasks as completed (checkbox)
- Swipe to delete tasks
- Save tasks locally using SharedPreferences
- Light & Dark mode (Material 3)
- Clean architecture (models, services, widgets, screens)
---
## Tech Stack
- **Flutter** (Material 3)
- **Dart**
- **SharedPreferences** (local storage)
- Clean folder structure (MVC-inspired)
